# Online-Quiz-Engine
🎯 Online Quiz Generator (Java GUI)
<p align="center"> <img src="https://media.giphy.com/media/3o7btPCcdNniyf0ArS/giphy.gif" width="250"/> </p> <p align="center"> <b>✨ Interactive Quiz Application with Java Swing ✨</b> </p>
🚀 Overview

The Online Quiz Generator is a GUI-based Java application built using Java Swing that allows users to create and attempt quizzes in an interactive and user-friendly interface.

Unlike traditional console apps, this project provides a modern UI experience with buttons, panels, and real-time feedback.

✨ Features

🎨 Graphical User Interface (GUI)

Clean and interactive design using Java Swing

📝 Quiz Creation

Add multiple-choice questions dynamically

✅ Real-Time Evaluation

Instant feedback on answers

📊 Score Tracking

Displays final score with performance

🔄 Retry Option

Attempt quiz again without restarting

⏱️ Timer Support (Optional Upgrade)

Adds challenge to quizzes

# Open project
cd online-quiz-gui

# Compile
javac Main.java

# Run
java Main
🧠 How It Works

💡 User launches the app
➡️ GUI window opens
➡️ Questions appear one by one
➡️ User selects answers via buttons
➡️ System evaluates instantly
➡️ Final score displayed in result screen
